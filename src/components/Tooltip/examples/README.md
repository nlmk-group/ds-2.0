# Tooltip examples

Эта папка содержит **внутренние примеры** для отладки компонента `Tooltip` (визуальная проверка, воспроизведение багов, регресс-проверки).

Важно:

- Не является публичной документацией и не предназначена для конечных пользователей дизайн-системы.
- Контент может меняться без соблюдения обратной совместимости.
- Используется исключительно командой разработки и поддержки.
